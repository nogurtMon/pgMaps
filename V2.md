# V2 Feature Roadmap

## Layers
- [ ] **PMTiles layer support** — add PMTiles URLs as static context layers alongside PostGIS layers (infrastructure, boundaries, parcels). MapLibre PMTiles plugin already supports this. Zero backend, CDN speed.
- [ ] **Raster layer support** — WMS/WMTS/XYZ raster tile layers as background or overlay
- [ ] **ArcGIS layer (live)** — render ArcGIS Feature Server as a live tile layer (not just import)

## Tile Performance
- [ ] **Pre-built Docker image on Docker Hub** — eliminate `--build` step, instant deploy
- [ ] **ST_Subdivide UI** — table management option to subdivide complex polygons into smaller pieces for dramatically faster tile queries on polygon layers (watersheds, parcels, boundaries)
- [ ] **Tile cache invalidation** — clear cache when a table is modified via the app

## Data & Editing
- [ ] **In-browser feature editing** — click a feature on the map and edit its attributes
- [ ] **Draw & insert** — draw points/lines/polygons directly on the map and insert them into PostGIS
- [ ] **Row delete from attribute table**

## Import
- [ ] **Scheduled ArcGIS sync** — re-scrape a Feature Server on a schedule and update the PostGIS table
- [ ] **PMTiles export** — generate a .pmtiles file from a PostGIS table for offline/CDN distribution

## Sharing & Collaboration
- [ ] **Password-protected share links** — optional password on `/share/[id]`
- [ ] **Embed code** — one-click `<iframe>` snippet for share links
- [ ] **Share link expiry** — set an expiration date on a share link

## UI & UX
- [ ] **Multi-layer attribute table** — switch between layers without closing the panel
- [ ] **Saved filter presets** — name and save a filter configuration per layer
- [ ] **Heatmap layer type** — density visualization for point datasets
- [ ] **3D extrusion** — extrude polygons by a numeric attribute (buildings, population)
- [ ] **Dark/light map theme toggle** — per-basemap theme preference saved to view
